## Installation

**Windows**

```plaintext
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

**Linux/mac**

```plaintext
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```
